PHP.Net.INET6Address.isGloballyRoutable ( )

Checks, if the address is globally routable.

These addresses are NOT routable:

  • ::1/128 – localhost
  • fe80::/10 – Link-local autoconf addresses
  • fc00::/7 – ULA
  • 2001:10::/28 – ORCHID (Overlay Routable Cryptographic Hash Identifiers)
  • 2001:db8::/32 – Documentation prefix

These addresses ARE routable:

  • 2000::/3 – Global unicast
  • ff0e::/16 – Global permanent multicast
  • ff1e::/16 – Global transient multicast
